# データポリシー

## 公開リポジトリに含めるもの

- アプリケーションコード
- DBスキーマと架空サンプル
- 汎用的なクローラー／インポート処理
- プレースホルダーだけを含む設定テンプレート

## 公開リポジトリに含めないもの

- 実在大学のPDF、画像、OCR結果、問題文、解答、解説
- PDFから作成したレビュー、ページ範囲、インポートSQL
- パスワードハッシュ、セッショントークン、学習履歴
- CloudflareアカウントID、D1/R2 ID、個人メール、本番ドメイン
- アクセス制限を伴う資料の取得情報

`data/crawl/`、`data/agent-review/`、`data/open-inshi/` はローカルまたは非公開ストレージ専用です。これらのパスは `.gitignore` と公開前チェックの両方で保護します。

## 第三者コンテンツ

大学が過去問を公開していても、再配布や改変が許可されているとは限りません。取得元へのリンクを保存する場合も、利用条件、著作権表示、公開期間、アクセス制限に従ってください。本リポジトリのMIT Licenseは第三者コンテンツへ適用されません。
